安装Docker

参考官方教程：https://docs.docker.com/get-docker/

windows

默认安装在C:\Program Files\Docker

解决方式：

在其它你希望的安装位置新建一个Docker文件，这里以 B:\Docker 为例

  1. 删除如下目录 C:\Program Files\Docker
  2. 以管理员身份打开cmd（win+R => 输入cmd => ctrl+shift+enter）
  3. cmd中执行命令（mklink /j “C:\Program Files\Docker” “B:\Docker”）
  4. 执行下载的安装程序

linux

卸载旧的版本

1
2
3
4

8

sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine

需要安装包

1

sudo yum install -y yum-utils

设置镜像仓库

官方镜像：（比较慢，不推荐）

1
2
3

sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo

阿里镜像仓库 ：（推荐）

1
2
3

sudo yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装 docker

安装前先更新yum软件包索引

1

yum makecache fast

安装docker-ce（社区版-免费的）

1

sudo yum install docker-ce docker-ce-cli containerd.io

启动docker

sudo systemctl start docker

判断是否成功安装docker 查看版本

docker version

因为使用阿里云镜像仓库，可以配置阿里云的镜像加速–速度更快（可选）

1.搜索阿里云

2.在阿里云里搜索 – 容器镜像服务

点击立即开通–镜像工具–镜像加速器–选择CentOS

使用CentOS中的代码配置

创建一个目录

sudo mkdir -p /etc/docker

编辑配置文件

1
2
3
4
5

sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["xxxx替换自己的加速地址"]
}
EOF

重启服务

sudo systemctl daemon-reload

重启docker

sudo systemctl restart docker

如果需要卸载docker

卸载依赖

 sudo yum remove docker-ce docker-ce-cli containerd.io

删除文件夹

1
2

 sudo rm -rf /var/lib/docker
 sudo rm -rf /var/lib/containerd
